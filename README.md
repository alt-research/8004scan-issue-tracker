# 8004scan Issue Tracker

Welcome to the official issue tracker for [8004scan](https://8004scan.io) - the AI Agent Explorer and Registry Platform built on the ERC-8004 standard.

## About 8004scan

8004scan is the definitive platform for discovering, exploring, and interacting with AI agents registered on the blockchain. Our platform provides:

- **Agent Discovery** - Search and filter AI agents by skills, domains, and capabilities
- **Agent Rankings** - 7-dimension scoring system with validator tier multipliers
- **Validation Network** - Multi-tier validator system for agent verification
- **X402 Integration** - Payment wallet metrics and financial activity tracking
- **Real-time Updates** - Live blockchain synchronization and notifications

## How to Use This Repository

This repository is dedicated to collecting community feedback, bug reports, and feature requests for 8004scan. We value your input and are committed to continuously improving the platform.

### Reporting Issues

Before creating a new issue, please:

1. **Search existing issues** to avoid duplicates
2. **Use the appropriate template** for your issue type
3. **Provide detailed information** to help us understand and address your concern

### Issue Types

| Type | Description | Template |
|------|-------------|----------|
| **Bug Report** | Something isn't working as expected | [Report a Bug](../../issues/new?template=bug_report.yml) |
| **Feature Request** | Suggest a new feature or enhancement | [Request Feature](../../issues/new?template=feature_request.yml) |
| **Feedback** | Share your experience or suggestions | [Give Feedback](../../issues/new?template=feedback.yml) |
| **Question** | Ask about usage or functionality | [Ask Question](../../issues/new?template=question.yml) |

### Issue Labels

We use the following labels to categorize and prioritize issues:

| Label | Description |
|-------|-------------|
| `bug` | Confirmed bugs or errors |
| `enhancement` | New features or improvements |
| `feedback` | General user feedback |
| `question` | Questions about the platform |
| `documentation` | Documentation improvements |
| `good first issue` | Good for newcomers |
| `help wanted` | Community help appreciated |
| `priority: critical` | Requires immediate attention |
| `priority: high` | Important, to be addressed soon |
| `priority: medium` | Normal priority |
| `priority: low` | Nice to have |
| `status: investigating` | Under investigation |
| `status: in-progress` | Being worked on |
| `status: resolved` | Issue has been resolved |
| `wontfix` | Will not be addressed |
| `duplicate` | Duplicate of another issue |

## Platform Components

8004scan consists of multiple components:

| Component | Description |
|-----------|-------------|
| **Web App** | Frontend application at [8004scan.io](https://8004scan.io) |

## Community Guidelines

Please review our [Code of Conduct](CODE_OF_CONDUCT.md) before participating. We are committed to maintaining a welcoming and inclusive community.

### Do's

- Be respectful and constructive
- Provide detailed, reproducible bug reports
- Search for existing issues before creating new ones
- Use appropriate issue templates
- Follow up on your issues when asked for more information

### Don'ts

- Post duplicate issues
- Include sensitive information (API keys, private keys, etc.)
- Engage in harassment or discrimination
- Spam or post off-topic content
